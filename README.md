
# KitchenCounter 

A restaurant inventory app to be used on mobile or tablet to make counting weekly inventories easier 


## Deployment

Add config.env file to config folder

Add to config.env
```bash
PORT = <preferred port>
MONGO_URI = <mongo uri string>
```

To Deploy:

```bash
  npm start
```


## Features

- Mobile Responsiveness
- User Login
- User Sessions



## Tech Stack

- Javascript
- Node
- Express
- EJS
- Mongoose
- Passport
